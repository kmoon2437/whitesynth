# 파라미터 목록
자체 sysex에서 쓰는 파라미터 목록이다.

## sysex 기본 구조
```
F0 7D 10 0B 15 rr rr rr rr vv ... F7
```
- `F0` = 이 메세지가 sysex 메세지임을 의미함
- `7D` = 제조사(manufacturer: 생산자) ID (테스트용) (자세한 것은 [여기](./sysexManufacturerId.md) 참조)
- `10` = Device ID (사실상 무쓸모지만 가급적 `10`으로 놔둘 것을 권장)
- `0B 15` = Model ID (whitesynth)
- `rr rr rr rr` = 파라미터 번호
- `vv ...` = 데이터(길이는 1바이트일 수도 있고 더 길 수도 있음)
- `F7` = sysex의 끝

## 분류
채널별 설정은 그냥 간단하게 20 0x nn nn으로 표기한다.

| 파라미터 번호 | 설명 |
| :---: | --------- |
| `00 00 nn nn` | 시스템 설정(리셋 sysex라던가) |
| `10 00 nn nn` | 기본 설정 |
| `11 00 nn nn` | 기본 이펙트 관련 설정 |
| `12 00 nn nn` | Insertion effect 관련 설정 |
| `20 00 nn nn` | 채널별 설정(1번) |
| `20 01 nn nn` | 채널별 설정(2번) |
| `20 02 nn nn` | 채널별 설정(3번) |
| : | ... |
| `20 0D nn nn` | 채널별 설정(14번) |
| `20 0E nn nn` | 채널별 설정(15번) |
| `20 0F nn nn` | 채널별 설정(16번) |

## 추가사항
「값 형태」는 보낼 값의 길이가 몇 바이트인지, 그 값의 어디를 읽고 어디를 무시할지에 대한 부분이다.  
이게 안 적혀 있는 등의 경우, 값의 길이나 읽는 방식이 상황에 따라 달라질 수 있음을 의미한다.(eg. Insertion effect 파라미터라던가)

공식을 적어 놓은 경우, 해당 공식은 일반 값(얘를 x라 함)을 sysex용 값으로 변환하는 공식이다.

파라미터별로 값의 길이가 정해져 있지만, 그보다 많이 보내도 적게 보내도 상관없다. 많이 보내면 뒷부분을 무시하고, 적게 보내면 명령을 무시한다.

## 목록

### 시스템 설정

#### System reset - `00 00 00 7F`
Reset 신호. 파라미터랍시고 넣어 놨지만 사실 파라미터라기보다는 명령이다. 따라서 값은 사실 없어도 된다.

- 값 형태: `0000 0000`
- 값: 0 (`0x00`)
- 기본값: 없음

### 기본 설정

#### Master tune - `10 00 00 00`
cent 단위로 전체 음정을 조율한다. 1키 = 100cent이다.

- 값 형태
  - `0kkk kkkk`
  - `0lll llll`
- 값: -100.0 - +100.0(cent) (공식: (10*x) + 1024)
- 기본값: 0

#### Master volume - `10 00 00 01`
최종 출력의 음량을 조절한다. 모든 이펙트가 적용된 뒤 가장 마지막에 적용되니 8820이나 SCVA를 쓴 적이 있으신 분들은 안심하시길.

- 값 형태
  - `0kkk kkkk`
- 값: 0 - 127
- 기본값: 127

#### Master key shift - `10 00 00 02`
전체 음정을 1키 단위로 조절한다.

- 값 형태
  - `0kkk kkkk`
- 값: -24 - +24 (공식: x + 64)
- 기본값: 0

#### Master pan - `10 00 00 03`
최종 출력의 좌우 음향을 조절한다.

- 값 형태
  - `0kkk kkkk`
- 값: L63(1) - Center(64) - R63(127)
- 기본값: Center

#### Reverb preset - `11 00 01 00`
이걸 설정하면 나머지 reverb 파라미터들이 자동으로 맞춰진다.

- 값 형태: `0kkk kkkk`
- 값: Room 1(0),Room 2(1),Room 3(2),Hall 1(3),Hall 2(4),Plate(5),Delay(6),Panning delay(7)
- 기본값: Hall 2

#### Reverb room type - `11 00 01 01`
reverb 이펙트용 방의 종류를 지정한다.

- 값 형태: `0kkk kkkk`
- 값: Room(0),Hall(1),Plate(2)
- 기본값: Hall(1)

설명: Room은 평범한 방(밴드 합주 연습실 정도),Hall은 흔한 공연장,Plate는 구식 철판 리버브다.

#### Reverb room size - `11 00 01 02`
reverb 이펙트용 방의 용적(크기)을 지정한다.

- 값 형태: `0kkk kkkk`
- 값: 0 - 127
- 기본값: ??

#### Reverb pre-lowpass-filter - `11 00 01 03`

- 값 형태: `0kkk kkkk`
- 값: 0 - 7
- 기본값: 4

#### Reverb level - `11 00 01 04`
reverb 출력음의 level을 설정한다.

- 값 형태: `0kkk kkkk`
- 값: 0 - 127
- 기본값: 64

#### Reverb time - `11 00 01 05`
reverb 이펙트로 인해 잔향이 남는 시간을 설정한다.

- 값 형태: `0kkk kkkk`
- 값: 0 - 127
- 기본값: 64

#### Reverb delay feedback - `11 00 01 06`
reverb 이펙트에서 delay로 인해 소리가 울리는 횟수를 지정한다.

- 값 형태: `0kkk kkkk`
- 값: 0 - 127
- 기본값: 0

#### Reverb pre-delay time - `11 00 01 07`
reverb 이펙트에서 소리가 울리기 시작하는 시간을 설정한다.

- 값 형태: `0kkk kkkk`
- 값: 0 - 127(ms)
- 기본값: 0

### Insertion effect 관련 설정

#### Insertion effect type - `12 00 1x 00`
이펙트 종류를 설정한다. 이걸 설정하면 모든 파라미터의 값이 이펙트별 기본값으로 지정된다.

- `x` = 이펙터 번호(`0x0` - `0xF`)
- 값 형태
  - `0kkk kkkk`
  - `0lll llll`
  - `0mmm mmmm`
- 값: `0x00 0x00 0x00` - `0xFF 0xFF 0xFF`
- 기본값: `0x00 0x00 0x00` (None)

#### Insertion effect parameter - `12 00 2x pp`
이펙트 파라미터를 설정한다.

- `x` = 이펙터 번호(`0x0` - `0xF`)
- `pp` = 파라미터 번호(`0x00` - `0x2F`)
- 값 형태
  - `0kkk kkkk`
  - ...이긴 하나 이게 몇 개가 될 지는 상황에 따라 다름
- 값: 상황에 따라 다름
- 기본값: 상황에 따라 다름

#### Insertion effect on/off - `12 00 3x 0h`
채널별 insertion effect on/off를 설정한다.

- `x` = 이펙터 번호(`0x0` - `0xF`)
- `h` = 미디 채널 번호(`0x0` - `0xF`)
- 값 형태
  - `0000 000k`
- 값: off(bypass: 그냥 통과)(0),on(1)
- 기본값: off
