## 구현 예정
- 일단 미디음원의 기본기능(note on/off, control change, pitch bend 같은 것들)
- sc 8820(사실상 scva)에서 지원하는것들 중 대강 70%는 만들 예정
- 인서션(insertion) 이펙트 1포트당 16개(8820이 1개밖에 지원 안해서 얘는 내친김에 16개까지 되게 만들 예정)

## 리빙 포인트(?)
- 오디오 샘플은 결국 백분율(?)이다. 당장 부동소수점 샘플의 최소/최대값이 -1.0 - +1.0이다.
  - 따라서 16비트 signed 샘플을 부동소수점으로 변환하려면 그냥 양수일 때 32767, 음수일 때 32768로 나누면 된다. ----- (1)
  - 24비트 signed 샘플도 마찬가지로 양수일 때 8388607, 음수일 때 8388608로 나누면 된다. ----- (2)
  - 즉 (1)을 해준 다음 (2)의 역연산을 해주면 16비트 샘플을 24비트 샘플로 변환할 수 있다.
- 정수 샘플 기준, unsigned 샘플은 그냥 signed 샘플에다가 최소값의 절대값을 더한 값이다.(최대값을 더하면 최소값에 더했을 때 -1이 나옴)
  - 부동소수점 샘플은 최소/최대값의 절대값이 1.0으로 같기 때문에 걍 1을 더하면 된다.
